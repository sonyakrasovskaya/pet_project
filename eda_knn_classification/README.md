# EDA & KNN Classification Research

Исследование данных и классификация с использованием метода k-ближайших соседей (KNN).

## Описание

Проект включает:
- Exploratory Data Analysis (EDA) - исследовательский анализ данных
- Реализацию и настройку алгоритма KNN для задачи классификации
- Сравнение различных параметров и метрик качества

## Файлы

- `EDA_research_&_KNN.ipynb` - основной ноутбук с EDA и реализацией KNN

## Методы

- K-Nearest Neighbors (KNN)
- Оптимизация гиперпараметров
- Оценка качества классификации
