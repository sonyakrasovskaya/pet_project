# Gradient Descent Optimization

Исследование различных методов оптимизации градиентного спуска.

## Описание

Проект посвящён изучению и сравнению различных алгоритмов оптимизации градиентного спуска для задач машинного обучения.

## Файлы

- `Gradient_Descent_optimization.ipynb` - основной ноутбук с реализацией и сравнением методов

## Методы

- Стандартный градиентный спуск
- Стохастический градиентный спуск (SGD)
- Mini-batch градиентный спуск
- Методы с адаптивным learning rate
